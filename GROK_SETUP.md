# Настройка Grok API

Проект теперь использует **Grok (xAI)** в качестве основного AI провайдера вместо GigaChat.

## Получение API ключа

1. Перейдите на [console.x.ai](https://console.x.ai)
2. Зарегистрируйтесь или войдите в аккаунт
3. Создайте новый API ключ в разделе API Keys
4. Скопируйте ключ

## Настройка проекта

Добавьте ключ в файл `.env`:

```env
# Grok (xAI) - Main AI Provider
GROK_API_KEY=your_grok_api_key_here
```

## Доступные модели

Проект автоматически выбирает модель в зависимости от задачи:

- **grok-2-mini** - для быстрых задач и чата (по умолчанию)
- **grok-2-latest** - для сложных задач, JSON генерации и больших ответов

## Fallback провайдеры

Если Grok недоступен, система автоматически переключится на:

1. **Groq** (LLaMA модели) - требует `GROQ_API_KEY`
2. **Gemini** (Google) - требует `GEMINI_API_KEY`

## Преимущества Grok

- Мощные модели с большим контекстом
- Быстрая генерация ответов
- Поддержка стриминга
- OpenAI-совместимый API
- Хорошее понимание русского языка

## Лимиты

Проверьте актуальные лимиты на [console.x.ai](https://console.x.ai):
- Rate limit: ~60 запросов/минуту (настраивается в коде)
- Токены зависят от вашего плана

## Миграция с GigaChat

Все упоминания GigaChat были заменены на Grok:
- ✅ Обновлён `lib/ai-router.ts`
- ✅ Обновлён `.env` и `.env.example`
- ✅ Удалены зависимости от `uuid` и `https` модулей
- ✅ Упрощена аутентификация (простой Bearer token)

Старые переменные окружения (`GIGACHAT_AUTH_KEY`) больше не используются и могут быть удалены.
