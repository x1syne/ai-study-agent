## 1. 📝 ГОТОВЫЙ КОД НА ЯЗЫКЕ 1С

### Автозаполнение цены материала из регистра сведений
```bsl
// Модуль формы документа ЗаявкаНаРемонт
// Вызывается при изменении поля "Материал" в табличной части

&НаКлиенте
Процедура МатериалыМатериалПриИзменении(Элемент)
    СтрокаТЧ = Элементы.Материалы.ТекущиеДанные;
    МатериалыМатериалПриИзмененииНаСервере(СтрокаТЧ.ПолучитьИдентификатор());
КонецПроцедуры

&НаСервере
Процедура МатериалыМатериалПриИзмененииНаСервере(ИдСтроки)
    СтрокаТЧ = Объект.Материалы.НайтиПоИдентификатору(ИдСтроки);
    Если СтрокаТЧ = Неопределено Тогда Возврат; КонецЕсли;
    
    Если НЕ ЗначениеЗаполнено(СтрокаТЧ.Материал) Тогда
        Возврат;
    КонецЕсли;
    
    // Запрос к регистру сведений ЦеныМатериалов
    Запрос = Новый Запрос;
    Запрос.Текст =
        "ВЫБРАТЬ ПЕРВЫЕ 1
        |    ЦеныМатериалов.Цена КАК Цена
        |ИЗ
        |    РегистрСведений.ЦеныМатериалов КАК ЦеныМатериалов
        |ГДЕ
        |    ЦеныМатериалов.Материал = &Материал";
    Запрос.УстановитьПараметр("Материал", СтрокаТЧ.Материал);
    
    РезультатЗапроса = Запрос.Выполнить();
    Если НЕ РезультатЗапроса.Пустой() Тогда
        Выборка = РезультатЗапроса.Выбрать();
        Выборка.Следующий();
        СтрокаТЧ.Цена = Выборка.Цена;
    КонецЕсли;
    
    // Пересчёт суммы строки
    СтрокаТЧ.Сумма = СтрокаТЧ.Количество * СтрокаТЧ.Цена;
    РассчитатьИтоговуюСтоимость();
КонецПроцедуры
```

### Расчёт итоговой стоимости заявки
```bsl
// Считает: стоимость вида работы + сумма всех материалов

&НаСервере
Процедура РассчитатьИтоговуюСтоимость()
    // Стоимость вида работы
    СтоимостьРаботы = 0;
    Если ЗначениеЗаполнено(Объект.ВидРаботы) Тогда
        СтоимостьРаботы = Объект.ВидРаботы.Стоимость;
    КонецЕсли;
    
    // Сумма по материалам
    СуммаМатериалов = 0;
    Для Каждого Строка Из Объект.Материалы Цикл
        Строка.Сумма = Строка.Количество * Строка.Цена;
        СуммаМатериалов = СуммаМатериалов + Строка.Сумма;
    КонецЦикла;
    
    // Итого
    Объект.ИтоговаяСтоимость = СтоимостьРаботы + СуммаМатериалов;
КонецПроцедуры

// Вызов при изменении количества в ТЧ:
&НаКлиенте
Процедура МатериалыКоличествоПриИзменении(Элемент)
    СтрокаТЧ = Элементы.Материалы.ТекущиеДанные;
    СтрокаТЧ.Сумма = СтрокаТЧ.Количество * СтрокаТЧ.Цена;
    РассчитатьИтоговуюСтоимостьНаСервере();
КонецПроцедуры
```

## 4. 📥 ИМПОРТ ЗАЯВОК ИЗ EXCEL

```bsl
// Обработка "ЗагрузкаЗаявокИзExcel"

    // Ожидаемые колонки Excel (строка 1 — заголовок):

        Если НЕ ЗначениеЗаполнено(ЗначениеДаты) Тогда
            Продолжить; // Пустая строка

        Попытка
            НоваяЗаявка = Документы.ЗаявкаНаРемонт.СоздатьДокумент();

            Телефон = СокрЛП(ТабДок.Область(НомерСтроки, 4).Значение);
```

## 5. 🔒 ИСТОРИЯ ИЗМЕНЕНИЙ ЗАЯВКИ

```bsl
// В модуле объекта документа ЗаявкаНаРемонт
// Событие: ПередЗаписью

Процедура ПередЗаписью(Отказ, РежимЗаписи, РежимПроведения)
    // Для новых документов история не нужна
    Если ЭтоНовый() Тогда Возврат; КонецЕсли;
    
    // Сравниваем с сохранёнными данными
    СтарыйОбъект = Ссылка.ПолучитьОбъект();
    
    // Проверяем изменение статуса
    Если СтарыйОбъект.Статус <> Статус Тогда
        ЗаписатьИсторию(Ссылка, "Статус",
            Строка(СтарыйОбъект.Статус),
            Строка(Статус));
    КонецЕсли;
    
    // Проверяем изменение мастера
    Если СтарыйОбъект.Мастер <> Мастер Тогда
        ЗаписатьИсторию(Ссылка, "Мастер",
            Строка(СтарыйОбъект.Мастер),
            Строка(Мастер));
    КонецЕсли;
    
    // Проверяем изменение итоговой стоимости
    Если СтарыйОбъект.ИтоговаяСтоимость <> ИтоговаяСтоимость Тогда
        ЗаписатьИсторию(Ссылка, "ИтоговаяСтоимость",
            Формат(СтарыйОбъект.ИтоговаяСтоимость, "ЧДЦ=2"),
            Формат(ИтоговаяСтоимость, "ЧДЦ=2"));
    КонецЕсли;
КонецПроцедуры

Процедура ЗаписатьИсторию(ЗаявкаСсылка, Поле, СтароеЗначение, НовоеЗначение)
    НЗапись = РегистрыСведений.ИсторияИзменений.СоздатьМенеджерЗаписи();
    НЗапись.Заявка          = ЗаявкаСсылка;
    НЗапись.ДатаИзменения   = ТекущаяДатаСеанса();
    НЗапись.Пользователь    = Строка(ПараметрыСеанса.ТекущийПользователь);
    НЗапись.ИзменённоеПоле  = Поле;
    НЗапись.СтароеЗначение  = СтароеЗначение;
    НЗапись.НовоеЗначение   = НовоеЗначение;
    НЗапись.Записать();
КонецПроцедуры
```

## 6. 🔍 ФИЛЬТРАЦИЯ СПИСКА ЗАЯВОК ПО СТАТУСУ

```bsl
// В модуле формы списка документа ЗаявкаНаРемонт

// Реквизит формы: ФильтрСтатус (СправочникСсылка.Статусы)

&НаКлиенте
Процедура ФильтрСтатусПриИзменении(Элемент)
    УстановитьФильтрПоСтатусу();

&НаСервере
Процедура УстановитьФильтрПоСтатусу()
    // Очищаем предыдущий фильтр
    Список.Отбор.Элементы.Очистить();
    
    Если ЗначениеЗаполнено(ФильтрСтатус) Тогда
        // Добавляем фильтр по выбранному статусу
        ЭлементОтбора = Список.Отбор.Элементы.Добавить(Тип("ЭлементОтбораКомпоновкиДанных"));
        ЭлементОтбора.ЛевоеЗначение  = Новый ПолеКомпоновкиДанных("Статус");
        ЭлементОтбора.ВидСравнения   = ВидСравненияКомпоновкиДанных.Равно;
        ЭлементОтбора.ПравоеЗначение = ФильтрСтатус;
        ЭлементОтбора.Использование  = Истина;
```
