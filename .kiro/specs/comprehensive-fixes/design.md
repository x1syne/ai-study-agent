# Комплексные исправления платформы

## Проблемы и решения

### 1. Теория — повторы "Введение", отсутствие реальных примеров

**Проблема:** Генерируемая теория содержит повторяющиеся секции "Введение", абстрактные примеры без связи с реальностью.

**Решение:**
- Обновить промпты в `lib/ai/prompts.ts` и `lib/ai/domain-prompts.ts`
- Добавить инструкции: "НЕ повторяй заголовок Введение", "Используй структуру Road.sh"
- Требовать конкретные примеры из индустрии (AI/ML, геймдев, финтех)

### 2. Практика — тривиальные задачи

**Проблема:** Генерируемые задачи слишком простые, не соответствуют уровню учебника высшей математики.

**Решение:**
- Обновить промпты генерации практики для MATHEMATICS домена
- Добавить требования: доказательства теорем, сложные вычисления, применение теорем
- Указать уровень сложности: "уровень учебника высшей математики"

### 3. LaTeX — неправильный рендеринг (ϵ → ε)

**Проблема:** Символы LaTeX отображаются некорректно (ϵ вместо ε).

**Решение:**
- Проверить компонент `TheoryContent.tsx` на правильную обработку LaTeX
- Убедиться что используется правильная библиотека (KaTeX/MathJax)
- Добавить CSS для корректного отображения греческих символов

### 4. Навигация — перезагрузка страницы

**Проблема:** При переходе между темами происходит полная перезагрузка страницы.

**Решение:**
- Использовать `next/link` вместо `router.push` где возможно
- Добавить `prefetch` для предзагрузки
- Использовать shallow routing для сохранения состояния

### 5. Вкладки "Модули/Все темы" — проверить работу

**Проблема:** Возможные баги в переключении вкладок на странице графа.

**Решение:**
- Проверить состояние `viewMode` в `graph/page.tsx`
- Убедиться что данные загружаются корректно для обоих режимов

### 6. Повторная генерация при переходе из "Мои курсы"

**Проблема:** При переходе на курс из вкладки "Мои курсы" начинается повторная генерация контента.

**Решение:**
- Добавить проверку кеша перед генерацией в `ModuleTopics.tsx`
- Проверять наличие существующего контента в БД
- Использовать `stale-while-revalidate` паттерн

## Файлы для изменения

1. `lib/ai/prompts.ts` — основные промпты
2. `lib/ai/domain-prompts.ts` — доменные промпты  
3. `components/learning/TheoryContent.tsx` — рендеринг LaTeX
4. `components/graph/ModuleTopics.tsx` — навигация и кеширование
5. `app/(dashboard)/graph/page.tsx` — вкладки
6. `app/api/modules/[id]/topics/route.ts` — API проверки кеша
