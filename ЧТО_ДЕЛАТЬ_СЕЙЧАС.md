# 🎯 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС — пошагово

## Твоя текущая ситуация:
✅ Проект создан: «service::Сервисный центр 1.0»
✅ Есть подсистема «Основное»
❌ Ещё нет ни одного объекта метаданных

---

## ШАГ 1 — СОЗДАЙ 5 СПРАВОЧНИКОВ (делай по очереди)

### Как создать справочник в 1С:Элемент:
1. Правой кнопкой на папке **«Основное»** → **«Добавить»** → **«Справочник»**
2. Задай имя
3. Добавь реквизиты на вкладке «Реквизиты»
4. Сохрани (Ctrl+S)

---

### 📁 СПРАВОЧНИК №1 — «Клиенты»
**Имя объекта:** `Клиенты`
**Реквизиты добавить:**
| Имя реквизита | Тип | Длина |
|---|---|---|
| Телефон | Строка | 20 |
| Email | Строка | 100 |

*(Поле "Наименование" уже есть по умолчанию — это и есть ФИО)*

---

### 📁 СПРАВОЧНИК №2 — «Техника»

|---|---|---|

| ГодВыпуска | Число | Длина=4, Точность=0 |

*(Поле "Наименование" = Модель)*

---

*(Поле "Наименование" = название вида работы)*

---

## ШАГ 2 — СОЗДАЙ РЕГИСТР СВЕДЕНИЙ «ЦеныМатериалов»

### Как создать регистр сведений:
1. Правой кнопкой на **«Основное»** → **«Добавить»** → **«Регистр сведений»**
2. Имя: `ЦеныМатериалов`
3. Вкладка **«Измерения»** → Добавить:
   - Имя: `Материал`, Тип: `СправочникСсылка.Материалы`
4. Вкладка **«Ресурсы»** → Добавить:
   - Имя: `Цена`, Тип: `Число`, Длина=15, Точность=2

---

## ШАГ 3 — СОЗДАЙ ДОКУМЕНТ «ЗаявкаНаРемонт»

### Как создать документ:
1. Правой кнопкой на **«Основное»** → **«Добавить»** → **«Документ»**
2. Имя: `ЗаявкаНаРемонт`

### Реквизиты шапки (вкладка «Реквизиты»):
| Имя | Тип |
|---|---|
| ДатаОкончания | Дата |
| Наименование | Строка (200) |
| Статус | СправочникСсылка.Статусы |
| Клиент | СправочникСсылка.Клиенты |
| Техника | СправочникСсылка.Техника |
| ВидРаботы | СправочникСсылка.ВидыРабот |
| Описание | Строка (неогр.) |
| ИтоговаяСтоимость | Число (15, 2) |
| Мастер | Строка (100) ← потом заменишь на пользователя |

### Табличная часть (вкладка «Табличные части»):
1. Нажми «Добавить табличную часть», имя: `Материалы`
2. Добавь колонки:
| Имя | Тип |
|---|---|
| Материал | СправочникСсылка.Материалы |
| Количество | Число (15, 3) |
| Цена | Число (15, 2) |
| Сумма | Число (15, 2) |

---

## ШАГ 4 — НАСТРОЙ ФОРМЫ

После создания документа:
1. Вкладка **«Формы»** → нажми **«+»** → создай **Форму документа**
2. На форме должны быть все реквизиты шапки
3. Добавь таблицу для ТЧ «Материалы»
4. Добавь кнопку **«Рассчитать стоимость»**

---

## ШАГ 5 — ДОБАВЬ КОД В ФОРМУ ДОКУМЕНТА

В модуле формы документа вставь код из файла `OLIMPIADA_1C_HELP.md`:
- Процедуру `МатериалыМатериалПриИзменении` — автозаполнение цены
- Процедуру `РассчитатьИтоговуюСтоимость` — расчёт итога
- Процедуру `СтатусПриИзменении` — email при завершении (позже)

---

## ШАГ 6 — СОЗДАЙ ФОРМУ СПИСКА ЗАЯВОК С ФИЛЬТРАЦИЕЙ

1. В документе `ЗаявкаНаРемонт` → вкладка **«Формы»** → **Форма списка**
2. Добавь реквизит формы: `ФильтрСтатус` (тип: СправочникСсылка.Статусы)
3. Добавь выпадающий список для этого реквизита над таблицей
4. Вставь код фильтрации из `OLIMPIADA_1C_HELP.md`

---

## ШАГ 7 — ПРАВА ДОСТУПА (⭐ 30 БАЛЛОВ!)

1. В дереве проекта найди раздел **«Роли»**
2. Создай роль **«Администратор»** — дай полный доступ ко всем объектам
3. Создай роль **«Мастер»**:
   - Справочники: только **Чтение**
   - Документ ЗаявкаНаРемонт: Чтение + Добавление + Изменение
   - Регистр ЦеныМатериалов: только Чтение
4. Для роли «Мастер» в документе → **Ограничение доступа**:
   ```
   ГДЕ ЗаявкаНаРемонт.Мастер = &ТекущийПользователь
   ```

---

## ШАГ 8 — ДОБАВЬ ПРИКРЕПЛЕНИЕ ФАЙЛОВ

1. Правой кнопкой на **«Основное»** → **«Добавить»** → **«Хранилище файлов»** (или «Справочник с файлами»)
2. Имя: `ФайлыЗаявок`
3. Привяжи к документу `ЗаявкаНаРемонт` через реквизит-владелец
4. Добавь на форму документа кнопку «Прикрепить файл»

---

- 10-15 заявок в разных статусах
