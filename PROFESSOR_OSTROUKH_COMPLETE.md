### 3. RAG система ✅
- **Векторная БД**: PostgreSQL + pgvector
- **Embeddings**: OpenAI text-embedding-ada-002 (1536 размерность)
- **Chunking**: 500 символов с перекрытием 2 предложений
- **Поиск**: Семантический поиск с порогом релевантности 0.25
- **Автоматическая интеграция**: При выборе персонажа "Проф. Остроух" система автоматически ищет релевантный контекст

## 🚀 Как запустить

### Шаг 1: Проверка окружения

Убедитесь, что в `.env` есть:
```env
# OpenAI для embeddings
OPENAI_API_KEY=your_key_here

# PostgreSQL с pgvector
DATABASE_URL=postgresql://user:password@localhost:5432/dbname

# Опционально: пропуск авторизации для разработки
SKIP_AUTH=true
NODE_ENV=development
```

### Шаг 2: Проверка БД

Убедитесь, что в PostgreSQL есть расширение pgvector и таблица documents:

```sql
-- Проверка расширения
SELECT * FROM pg_extension WHERE extname = 'vector';

-- Проверка таблицы
\d documents

-- Должна быть колонка: embedding vector(1536)
```

## 🔧 Troubleshooting

### Проблема: "No results found"

**Причины:**
1. База не проиндексирована
2. Запрос не релевантен темам профессора
3. Порог релевантности слишком высокий

**Решение:**
```bash
# 1. Проверьте индексацию
curl -X PUT http://localhost:3000/api/professor/knowledge

# 2. Проверьте БД
psql $DATABASE_URL -c "SELECT COUNT(*) FROM documents WHERE metadata->>'source' = 'ostroukh';"

# 3. Снизьте порог в коде (professor-knowledge.ts)
threshold: 0.15  // вместо 0.25
```

### Проблема: "Embeddings generation failed"

**Причины:**
1. Неверный OPENAI_API_KEY
2. Нет доступа к API OpenAI
3. Превышен лимит запросов

**Решение:**
```bash
# Проверьте ключ
echo $OPENAI_API_KEY

# Проверьте доступ
curl https://api.openai.com/v1/models \
  -H "Authorization: Bearer $OPENAI_API_KEY"
```

### Проблема: "Database connection failed"

**Причины:**
1. PostgreSQL не запущен
2. Неверный DATABASE_URL
3. Нет расширения pgvector

**Решение:**
```bash
# Проверьте PostgreSQL
pg_isready

# Проверьте подключение
psql $DATABASE_URL -c "SELECT version();"

# Установите pgvector
psql $DATABASE_URL -c "CREATE EXTENSION IF NOT EXISTS vector;"
```
