# Настройка RAG системы

## Быстрый старт

### 1. Выполни SQL в Supabase

1. Открой [Supabase Dashboard](https://supabase.com/dashboard)
2. Выбери проект → SQL Editor
3. Скопируй и выполни:
   ```
   prisma/migrations/add_vector_search.sql
   ```

### 2. Добавь переменные в Vercel

```env
SUPABASE_SERVICE_ROLE_KEY=твой_service_role_key
HUGGINGFACE_API_KEY=опционально_для_большего_лимита
SERPER_API_KEY=опционально_для_google_search
```

## Архитектура RAG

```
┌─────────────────────────────────────────────────────────────┐
│                    getFullRAGContext()                       │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐       │
│  │   Vector DB  │  │  Wikipedia   │  │    arXiv     │       │
│  │  (pgvector)  │  │   (free)     │  │   (free)     │       │
│  └──────────────┘  └──────────────┘  └──────────────┘       │
│                                                              │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐       │
│  │   Serper     │  │  DuckDuckGo  │  │ Open Library │       │
│  │  (2500/mo)   │  │   (free)     │  │   (free)     │       │
│  └──────────────┘  └──────────────┘  └──────────────┘       │
│                                                              │
│  ┌──────────────────────────────────────────────────┐       │
│  │              User Context (персонализация)        │       │
│  │  - Пройденные темы                                │       │
│  │  - Уровень пользователя                           │       │
│  │  - Связанные темы курса                           │       │
│  └──────────────────────────────────────────────────┘       │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│                         CACHE                                │
│  Memory (5 min) → Supabase (configurable TTL)               │
└─────────────────────────────────────────────────────────────┘
```

## Улучшения

| Функция | Описание |
|---------|----------|
| **Кэширование** | 2 уровня: память + Supabase |
| **Дедупликация** | Хэш контента предотвращает дубликаты |
| **Таймауты** | 5s на источник, 10s общий |
| **Ранжирование** | Vector → Wiki → arXiv → Web |
| **Персонализация** | Учёт истории обучения |
| **Автоиндексация** | Новый контент сохраняется в базу |

## Использование

```typescript
import { getFullRAGContext } from '@/lib/rag'

// Базовый RAG
const context = await getRAGContext('Python', 'Программирование')

// Полный RAG с персонализацией
const fullContext = await getFullRAGContext('ООП', 'Python', userId)
```
